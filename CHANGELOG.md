# Changelog

## 1.0.0 — 2026-07-22

Перший стабільний публічний реліз Metro Kyiv.

### Основне

- планування маршрутів між усіма 52 чинними станціями;
- перевірка 5 304 напрямків маршрутів у CI;
- інтерактивна схема з підсвічуванням маршруту;
- сторінки станцій, обрані станції та розрахункові таймери;
- туристичні місця Києва українською та англійською;
- геолокація найближчої станції;
- встановлення як PWA та офлайн-режим;
- світла, темна й системна тема;
- поширення маршруту через URL.

### Якість і доступність

- клавіатурна навігація та skip-link;
- керування фокусом у модальних вікнах;
- підтримка `prefers-reduced-motion` і forced colors;
- доступні контрасти та сенсорні області;
- Lighthouse CI для маршруту, схеми, місць і сторінки станції;
- фінальні результати: Accessibility 100, Best Practices 96, SEO 100.

### Релізна інфраструктура

- типізація, data validation, production build і bundle budgets у GitHub Actions;
- машинно-читаний `version.json` із версією, комітом і часом збірки;
- production smoke-перевірка HTML, ресурсів, маніфесту, Service Worker і deep links;
- автоматична перевірка після GitHub Pages deployment;
- GitHub Release із ZIP production-збірки та SHA-256 за тегом `v1.0.0`.
